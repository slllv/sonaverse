The Sonaverse

The Sonaverse is a liminal space where all virtual characters are meant to reside. As the Sonaverse grows, more new exciting features will be added, in order to allow creators and characters to interact within this space! We will start by adding recognized original characters from independent creators, but as the site grows, we plan on adding authentication features and character submission forms, as well as exciting games!



Contact

You can contact us to request a new character to be added, or ask any questions. we will give you a reply as soon as we can! (Feel free to resend your mail if you get no response within two days!)


Support

Sonaverse is a passion project made by one person, but as everything, it has associated costs. These include basic maintenance, updates and hosting/domain costs. If you believe in this project, a small tip is really appreciated!

This website is always a work a progress. feel free get in touch and make the Sonaverse a good place for everyone!
